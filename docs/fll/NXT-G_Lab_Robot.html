NXT-G Lab Robot

Build a simple robot quickly from the parts in the base Lego Mindstorms NXT kit.

The build diagrams are not as complete as those in a typical Lego kit.

The completed robot could be used in a FLL competition, but would not be competitive. That is by design.

For help with part names, consult the NXT Mindstorm kit guide.

Note: plans for a similar robot built from the parts in the Lego Mindstorms Education NXT kit are here.

Unable to see images of the build steps? Be sure to allow javascript in your browser.

----- Wiring ----
Assume the light sensor is on the front of the robot.
Wire the left motor to B.
Wire the right motor to C.
Wire the touch sensor to 1.
Wire the light sensor to 2.
